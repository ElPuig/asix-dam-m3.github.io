Kotlin és un llenguatge 100% compatible amb el JDK que proporciona una sintaxi més concisa i expressiva.

Un dels camps en els quals més s'utilitza Kotlin és per al desenvolupament d'apliacions Android.

En aquest CodeLab veurem un petita iniciació al llenguatge Kotlin, fent una comparació amb Java.

Inicia IntelliJ i crea un nou projecte anomenat "kotlin".

Selecciona "Kotlin/JVM" en el Tipus de projecte

Comencem amb Kotlin creant un nou fixter de codi font.

A la carpeta src del projecte fes click-dreta i selecciona New > Kotlin File/Class

Posa-li el nom Main

Pot ser et sorprén veure el fitxer en blanc, sense cap classe definida. En Kotlin no són obligatòries les classes.

Afegim el següent codi, el clàssic HelloWorld

fun main(args: Array<String>){
   println("Hello kotlin")
}

Executa el programa fent click-dreta i seleccionant "Run ‘MainKt'"

Les variables en Kotlin són exactament igual que en Java, però amb una sintaxi més concisa: no és necessari declarar el tipus de les variables:

var enter = 0;
var decimal = 5.4
var llarg = 2423_4327_6827_3625_423L
var boolea = true
var caracter = 'a'
var text = "kotlin mooola"

Els Strings incorporen l'ús de templates, és a dir, expressions de codi que són evaluades i concatenades a l'String.

var a = 3
var m = "moola"
var t = "kotlin $m $a ${a+1}"   // kotlin moola 3 4

També soporta els Strings multilínia:

var text2 = """
   aquest string ocupa
   varies
   linies
   """

var text3 = """
   |aquest string ocupa
   |varies
   |linies
   |sense sagnat
   """.trimMargin()

println(text2)
println(text3)

Kotlin incorpora una API per a entrada i sortida: kotlin.io

var n = readLine()
println("hola $n")

També pots usar la API de Java:

var scanner = Scanner(System.`in`)

var p = scanner.nextInt();
var q = scanner.nextFloat();

System.out.println(p)
System.out.format("%4.2f", q)

When

when substitueix l'operador de switch. En la forma més simple, és així:

var x = 4
when (x) {
   1 -> print("x == 1")
   2 -> print("x == 2")
   else -> {
       print("x no és 1")
       print("x no és 2")
   }
}

La coincidència de valors amb when és més expressiva:

var valor = 10;
var numeros = arrayOf(34,23,56,34)

when(valor){
   1 -> println("Valor es 1")
   2 -> {
       println("Valor no es 1")
       println("Valor es 2")
   }
   3,4 -> println("Valor es 3 o 4")
   in 5..10 -> println("valor es entre 5 i 10")
   in numeros -> println("valor es algun numero de l'array")
   else -> {
       println("Valor no es cap anterior")
       println("Valor es $valor")
   }
}

While

var n = 10
while(n < 20){
   println(n)
   n++
}

do {
   print(n)
   n--
} while (n > 0)

For

El for en Kotlin ofereix una forma molt còmoda de recorrer Arrays.

for(i in 0..10){  // for(i=0; i<10; i++)
   println(i)
}

for (i in 10 downTo 0){  // for(i=10; i>=10; i--)
   println("down   $i")
}


var llista = arrayOf(76,56,45,76)

for (i in 0..llista.size){
   println(llista[i])
}

for (i in llista.indices){
   println(llista[i])
}

for (numero in llista){
   println(numero)
}
