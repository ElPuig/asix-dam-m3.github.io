L'objectiu d'aquest CodeLab és aprendre el mètodes que ofereix JAVA per a la conversió entre tipus numèrics (int, float,...) i Strings.

Ja coneixes l'ús dels mètodes print i println per a impressió de cadenes a la sortida estàndard (System.out).

Atès que tots els números es poden convertir en cadenes, podeu utilitzar aquests mètodes per imprimir una barreja arbitrària de cadenes i nombres. El llenguatge de programació Java té altres mètodes, però, que us permeten exercir molt més control sobre la vostra impressió.

Sovint, un programa acaba amb dades numèriques en un objecte tipus String: un valor introduït per l'usuari, per exemple.

Podem convertir aquest String al seu valor numeric amb el mètode valueOf.

String sEnter = "10";
int enter = Integer.valueOf(sEnter);

String sDecimal = "10.99";
float decimal = Float.valueOf(sDecimal);

String sBoolea = "true";
boolean boolea = Boolean.valueOf(sBoolea);

Per convertir números a Strings hi ha varies opcions:

La primera és utilitzar la concatenació:

int numero = 1000;

String sNumero = ""+numero;

També es pot utilitzar el mètode valueOf de la classe String.

int numero = 1000;

String sNumero = String.valueOf(numero);

O també, el mètode toString de les classes numèriques:

int enter = 1000;
float decimal = 99.876f;

String sEnter = Integer.toString(enter);
String sDecimal = Float.toString(decimal);

Un cas especial és el tipus char, que es pot interpretar com un número (el seu codi) o com una lletra:

char caracter = 'a';

System.out.println(caracter); // com a lletra
System.out.println(2 + caracter);  // com a número

Podem forçar la interpretació d'un char com una lletra amb Character.toString(), i com un número amb Character.getNumericValue()

char caracter = 'a';

System.out.println(caracter);
System.out.println(">" + caracter);
System.out.println(2 + Character.toString(caracter));
System.out.println(">" + Character.getNumericValue(caracter));
