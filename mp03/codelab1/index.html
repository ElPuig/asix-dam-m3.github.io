El Java Development Kit proporciona eines de desenvolupament per a la creació de programes JAVA.

Què aprendràs

  • El procés de desnvolupament de programes amb tecnologia Java
  • Compilar un programa a partir del codi font
  • Executar un programa compilat

Descarrega el JDK

Trobarás el JDK oficial d'Oracle al següent enllaç:

Oracle Java Development Kit (JDK)

Important: Has de tenir en compte l'arquitectura de l'ordinador (x84/x64) i el Sistema Operatiu.

✅ Comprova que s'ha descarregat l'arxiu a la carpeta de baixades

Descomprimeix l'arxiu al teu directori personal

Obre l'arxiu descarregat amb el Gestor d'Arxivadors i extreu el contingut a la teva Carpeta Personal

Prova el compilador i l'intèrpret

Comprova que pots executar el compilador i l'intèrpret, fent que mostrin la seva versió. Per a fer-ho, obre un terminal i executa les següents comandes:

1 - Executa el compilador amb l'opción que mostra la versió:

jdk1.8.0_144/bin/javac -version

✅ Comprova que mostra la versió

2 - Executa l'intèrpret amb l'opció que mostra la versió:

jdk1.8.0_144/bin/java -version

✅ Comprova que mostra la versió

Si tot ha anat correctament, t'haurà sortit un resultat com aquest:

Anema a realitzar una petita aproximació al procés de desenvolupament que hem vist anteriorment.

En resum, el procés de desenvolupament és el següent:

  1. Escrivim el codi font en un archiu de text
  2. Compilem el codi font mitjançant el compilador javac, que el tradueix a bytecode.
  3. Executem el programa compilat amb la Java Virtual Machine java

Escriu el codi font del programa "Hola Mon"

Obre un editor de text sense format, com gedit, i crea un arxiu en la teva carpeta personal, amb el nom HelloWorldApp.java

Copia el següent codi i desa els canvis:

class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

Nota: Tecleja tot el codi, comandes i noms d'arxius, exactament com es mostren. Tant el compilador (javac) com el llançador (java) son case-sensitive, així que has de posar les majúscules correctament.

HelloWorldApp no és el mateix que helloworldapp

Compila l'arxiu amb el codi font

Utilitza el següent comandament al terminal per a compilar l'arxiu de codi font que tot just has creat:

jdk1.8.0_144/bin/javac HelloWorldApp.java

Aquest comandament haurà generat un fitxer anomenat HelloWorldApp.class amb el bytecode del teu programa. Comprova-ho.

Executa el programa amb la Màquina Virtual de Java

jdk1.8.0_144/bin/java HelloWorldApp

¡Felicitats! ¡Has executat el teu primer programa!
