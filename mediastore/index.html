The Media provider contains meta data for all available media on both internal and external storage devices.

https://developer.android.com/reference/android/provider/MediaStore

Definir los requestCode:

static final int RC_IMAGE_PICK = 9000;
static final int RC_VIDEO_PICK = 9001;
static final int RC_AUDIO_PICK = 9002;

Iniciar la Galería para seleccionar Image/Video/Audio

startActivityForResult(Intent(Intent.ACTION_PICK, android.provider.MediaStore.Images.Media.EXTERNAL_CONTENT_URI), RC_IMAGE_PICK);

startActivityForResult(Intent(Intent.ACTION_PICK, android.provider.MediaStore.Video.Media.EXTERNAL_CONTENT_URI), RC_IMAGE_PICK);

startActivityForResult(Intent(Intent.ACTION_PICK, android.provider.MediaStore.Audio.Media.EXTERNAL_CONTENT_URI), RC_IMAGE_PICK);
