Classes i objectes

Un objecte és una variable que té altres variables a dintre. És una forma d'agrupar variables relacionades entre sí.

Per exemple, si estem fent un programa que tracta amb productes, pot ser necessitem guardar, per exemple, els codis, els preus i l'stock d'aquests productes. Per a guardar el codi i el preu d'un únic producte podríem utilitzar aquestes dues variables:

String codi = "HWL900"; float preu = 99.5f; int stock = 10;

Ara bé, si volem emmagatzemar les dades de 5 productes, necessitaríem 10 variables, o dos arrays com aquests:

String[] codis = { "HWL900", "SW30", "ZX64", "iBnn", "A-10" }; float[] preus = { 99.5f, 64f, 39.5f, 40f, 9.99f }; int[] stocks = { 10, 30, 44, 8, 200 }; // TODOOO

Sharing references More than one variable can refer to the same object.

It is important to understand that two variables refer to the same data in memory rather than two independent copies. Since our class is mutable, we can modify the object using both references.
