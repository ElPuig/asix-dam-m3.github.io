El objetivo de esta práctica es trabajar con Fragments.

Un Fragment representa un comportamiento o una parte de la interfaz de usuario en una Activity. Puedes combinar múltiples fragmentos en una sola actividad para crear una IU multipanel y volver a usar un fragmento en múltiples actividades. Puedes pensar en un fragmento como una sección modular de una actividad que tiene su ciclo de vida propio, recibe sus propios eventos de entrada y que puedes agregar o quitar mientras la actividad se esté ejecutando (algo así como una "subactividad" que puedes volver a usar en diferentes actividades).

Orientación en runtime

Para obtener la orientación de la pantalla en runtime (es decir desde el código Java), haremos un pequeño truquito.

Añadiremos dos ficheros de recursos de valores que llamaremos config.xml, uno de ellos con el calificador landscape. Dentro de los ficheros añadiremos un recurso booleano el cual será false para la versión normal y true para la versión landscape.

Luego desde el código Java, podemos obtener el valor del booleano, que será true o false, en función del fichero que haya cargado (que dependerá de la orientación).

⏩Crea dos versiones alternativas del archivo config.xml

Sobre la carpeta 

res/values

selecciona New -> Values resource file. Ponle el nombre de fichero

En la versión normal añade el siguiente item:

   <item name="land" type="bool">false</item>
</resources>

En la versión land dale el valor

Comunicación entre Fragments

En este apartado veremos cómo se pueden pasar datos, un String por ejemplo, de un Fragment a otro.

Necesitamos que cuando se haga click en el botón, el MainFragment envíe un TEXTO al DetailFragment para que lo muestre en el TextView. La comunicación entre Fragments será diferente según la orientación:

  • Portrait: el MainFragment iniciará la DetailActivity, y le pasará le TEXTO en el Intent. El DetailFragment recuperará el Intent y de ahí sacará el TEXTO.
  • Landscape: el MainFragment ejecutará un método de la MainActivity y le pasará el TEXTO como parámetro. La MainActivity a su vez, ejecutará otro método del DetailFragment, pasándole el TEXTO.

Portrait

Añadiremos un listener al botón, para que inicie la DetailActivity, y le pase un texto en el Intent.

La DetailActivity incluye el DetailFragment, quien pondrá en el TextView el texto sacado del Intent.

public class MainActivity extends AppCompatActivity implements MainFragment.OnButtonClickListener{

   @Override
   protected void onCreate(Bundle savedInstanceState) {
       super.onCreate(savedInstanceState);
       setContentView(R.layout.activity_main);
   }

   @Override
   public void onButtonClick(String text) {
       if(isLandscape()) {
           DetailFragment fragment = (DetailFragment) getFragmentManager().findFragmentById(R.id.fragmentDetail);
           fragment.updateTextView(text);
       }else {
           Intent intent = new Intent(this, DetailActivity.class);
           intent.putExtra("TEXT", text);
           startActivity(intent);
       }
   }
}

public class MainFragment extends Fragment {

   interface OnButtonClickListener {
       void onButtonClick(String text);
   }

   OnButtonClickListener mListener; // el listener será la MainActivity

   @Override
   public View onCreateView(LayoutInflater inflater, ViewGroup container,
                            Bundle savedInstanceState) {
       super.onCreateView(inflater, container, savedInstanceState);
       View mView = inflater.inflate(R.layout.fragment_main, container, false);

       mView.findViewById(R.id.button).setOnClickListener(new View.OnClickListener() {
           @Override
           public void onClick(View view) {
               mListener.onButtonClick("Hello fragment communication!");
           }
       });
       return mView;
   }

   @Override
   public void onAttach(Context context) {
       super.onAttach(context);
       try {
           mListener = (OnButtonClickListener) context; // context==MainActivity
       } catch (ClassCastException e) {
           throw new ClassCastException(context.toString() + " must implement OnButtonClickListener");
       }
   }
}

      @Override

En la clase

  • Definimos el interfaz para el Listener del evento OnItemSelected
  • Declaramos abstracto el método que rellena el ArrayList, para que sean las subclases los que lo implementen
public abstract class ListFragment extends Fragment {

   public interface OnItemSelectedListener {
       void onItemSelected(String text);
   }

   OnItemSelectedListener mListener;
   private RecyclerView mRecycler;
   RecyclerAdapter mRecyclerAdapter;
   ArrayList<String> items = new ArrayList<>();

   @Override
   public View onCreateView(LayoutInflater inflater, ViewGroup container,
                            Bundle savedInstanceState) {
       super.onCreateView(inflater, container, savedInstanceState);
       View mView = inflater.inflate(R.layout.fragment_list, container, false);

       mRecycler = (RecyclerView) mView.findViewById(R.id.recyclerView);
       mRecycler.setLayoutManager(new LinearLayoutManager(getActivity()));
       mRecycler.setHasFixedSize(true);

       mRecyclerAdapter = new RecyclerAdapter(items);
       mRecycler.setAdapter(mRecyclerAdapter);

       populateList();

       return mView;
   }

   @Override
   public void onAttach(Context context) {
       super.onAttach(context);
       try {
           mListener = (OnItemSelectedListener) context;
       } catch (ClassCastException e) {
           throw new ClassCastException(context.toString() + " must implement OnItemSelectedListener");
       }
   }

   public abstract void populateList();

   class RecyclerAdapter extends RecyclerView.Adapter<RecyclerAdapter.ItemViewHolder> {

       private List<String> items;

       RecyclerAdapter(List<String> items) {
           this.items = items;
       }

       @Override
       public ItemViewHolder onCreateViewHolder(ViewGroup viewGroup, int i) {
           View view = LayoutInflater.from(viewGroup.getContext()).inflate(R.layout.item, viewGroup,false);
           return new ItemViewHolder(view);
       }

       @Override
       public void onBindViewHolder(final ItemViewHolder viewHolder, int i) {
           final String item = items.get(i);

           viewHolder.title.setText(item);

           viewHolder.itemView.setOnClickListener(new View.OnClickListener() {
               @Override
               public void onClick(View view) {
                   mListener.onItemSelected(item);
               }
           });
       }

       @Override
       public int getItemCount() {
           return (items != null ? items.size() : 0);
       }

       class ItemViewHolder extends RecyclerView.ViewHolder {
           private TextView title;

           ItemViewHolder(View view) {
               super(view);
               this.title = (TextView) view.findViewById(R.id.title);
           }
       }
   }
}
