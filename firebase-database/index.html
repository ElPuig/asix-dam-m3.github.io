Firebase Realtime Database es una base de datos alojada en la nube. Los datos se almacenan en formato JSON y se sincronizan en tiempo real con cada cliente conectado.

https://firebase.google.com/docs/database/android/start

Utilitza el projecte creat a la pràctica anterior (firebase-auth).

Afegeix la següent llibreria al build.gradle (Module: app)

implementation 'com.firebaseui:firebase-ui-database:4.3.1'

Accedeix a la consola: https://console.firebase.google.com

Crea la Realtime Database:

Activa-la en Mode de prova:

:

 
{
  "posts" : {
    "all-posts" :{
      $PostId : true,
    },
    "data" :{
      $PostId: {
        "author": String,
        "authorPhotoUrl": String,
        "content": String,
        "likes" : {
          $UserId: true
        },
        "mediaType":  "video" | "image" | "audio",
        "mediaUrl": String
      }
    },
    "user-likes": {
      $UserId: {
        $PostId: true
      }
    },
    "user-posts": {
      $UserId: {
        $PostId: true
      }
    }
  },
  "users": {
    $UserId : {
      "displayName": String,
      "email": String,
      "photoUrl": String
    }
  }
}
